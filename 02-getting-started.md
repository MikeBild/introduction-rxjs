# Getting Started

## Higher-Order Functions

### Unfold (aka create, generate)

`T -> Observable<T'>`

### Bind (aka map, projection, transform)

`Observable<T> -> Observable<T'>`

### Fold (aka reduce, accumulate, aggregate)

`Observable<T> -> T`

## Group of Operations

* Creation
* Reduction
* Inspection
* Aggregation
* Transformation
